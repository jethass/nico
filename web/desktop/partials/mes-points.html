Les différents niveaux

En cumulant des points, vous pouvez atteindre des niveaux dans votre sevrage tabagique :

  • j'ai fixé une date d'arrêt :
    Prêt pour le jour J

  • - de 500 points :
    Motivé pour réussir

  • entre 500 et 1500 points :
    Je tiens le cap

  • entre 1500 et 3000 points :
    Je continue dans cette voie là

    Je tiens le cap
